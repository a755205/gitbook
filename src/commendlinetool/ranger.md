# ranger

## description
- ranger is a terminal file manager with vim-like keybindings that uses the ncurses library to provide a powerful interface for your filesystem.

## installation on mac
```zsh
brew install ranger
```

## usage
```zsh
ranger
```

## keybindings


## cmd
- `:` is the command mode
- `:shell` - open shell in ranger

## commands
| description      | command         |
|------------------|-----------------|
| open folder      | `:shell open .` |
| change directory | `:cd`           |
